from numpy import log as ln # for ln
import math # sin

def function_a(x, y):
    return y * ln(x)

def function_b(x, y):
    return 2 * y * math.sin((x-1)/20)

Tworzę DataFrame'y, na podstawie których będę wyuczał algorytmy¶

Uyj objektu DataFrame z biblioteki pandas. jest on wspierany przez wikszo algorytmw, a zintegrowane wygodne operacje na zbiorze uatwi mi prac. Tworz wic dwa DataFrame:

  • df_a zawiera wszystkie kombinacje wartoci x i y z wygenerowanych zbiorw i wynik funkcji A dla nich
  • df_b zawiera wszystkie kombinacje wartoci x i y z wygenerowanych zbiorw i wynik funkcji B dla nich

from pandas import DataFrame # for dataframes

list_a = []

for x in list(x_set):
  for y in list(y_set):

    data['x'] = x
    data['y'] = y
    data['result'] = function_a(x,y)
    list_a.append(data)
    
df_a = DataFrame(list_a)
    
list_b = []

  1. Warstwa wejciowa skadajca si skadajca si z dwch wartoci (bo 2 zmienne) oraz z 4 neuronw, co oznacza, e warstwa musi nauczy si a 4 zmiennych wewntrzanych
  2. Warstwa ukryta skadajca si z 4 neuronw
  3. Warstwa wyjciowa, skadajca si z jednego neuronu
In [17]:
import pandas as pd
from keras.models import Sequential
from keras.layers import Dense
import numpy as np
from sklearn.model_selection import train_test_split
import time

start = time.time()

model = Sequential()
model.add(Dense(4, input_dim=2, activation='sigmoid'))
model.add(Dense(4))
model.add(Dense(1))

Optymalizator (funkcja optymalizacyjna) polega na takim dopasowaniu zmiennych wewntrznych, aby jak najbardziej zmniejszy warto funkcji straty. Wspczynnik uczenia informuje algorytm, jak agresywnie powinien stara si znale najlepszy model.

Wykorzystuj optymalizator Adam. Wykorzystuj rwnie funkcj straty mean_squared_error. Funkcja straty okrela, jak daleko przewidziana warto znajduje si od podanej wartoci. Wykorzystana przeze mnie funkcja jest redni bdu kwadratowego wyraanej jako suma kwadratu rnicy midzy przewidywanymi i rzeczywistymi zmiennymi docelowymi dla wszystkich punktw danych, podzielona przez liczb tych punktw, (wzr poniej).

In [18]:
model.compile(loss='mean_squared_error', optimizer='adam', metrics=['accuracy'])

Teraz naley wytrenowa model przy uyciu metody fit. Metoda przyjmuje parametr epochs, ktry okrela ilo penych przej przez peen zbir - np. dla wartoci 300 nasz model zostanie wytrenowany 300 * 812 = 243600 przykadami. Poniej zobaczymy rwnie, ile zajo pene wykonanie algorytmu, od momentu okrelania warstw uczcych.
